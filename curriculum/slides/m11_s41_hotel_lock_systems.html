(41-1)

Title Slide - Access Control Systems

  • Welcome to Access Control System engineering.
  • Learn how smart card readers and locks secure doors.
  • Understand door controllers, request-to-exit buttons, and relays.
(41-2)

RFID Principles (125kHz vs 13.56MHz)

  • 125kHz proximity cards send simple unencrypted ID numbers.
  • 13.56MHz Smart Cards (Mifare) use encrypted data chips for high security.
  • Select card technology based on site security requirements.
(41-3)

Wiegand Hardware Interface

  • Wiegand protocol uses 3 wires: Data 0 (Green), Data 1 (White), Ground.
  • Data 0 pulses low for binary 0; Data 1 pulses low for binary 1.
  • Maximum Wiegand cable distance is 150 meters using shielded wire.
(41-4)

Controller Architecture

  • Access Controllers store card databases and door logic locally.
  • Doors stay operational even if network connection to server breaks.
  • 1-Door, 2-Door, and 4-Door control panel options.
(41-5)

Request to Exit (REX) Loops

  • REX button allows users to unlock the door from inside to exit.
  • IR Motion REX sensors detect approaching people automatically.
  • Wire REX to controller input terminals for proper exit logging.
(41-6)

Electromagnetic Lock NC Loops

  • Magnetic Locks require constant power to stay locked (Fail-Safe).
  • Cutting power instantly releases the door magnet for exit.
  • Connect Maglocks through Normally Closed (NC) relay terminals.
(41-7)

Fire Integration Overrides

  • Fire Alarm integration MUST unlock all emergency exit doors automatically.
  • Wire fire alarm relay directly into the main lock power supply.
  • Provides hardware-level safety even if the access panel freezes.
(41-8)

Management Software Databases

  • Access control software manages user card profiles and permissions.
  • Set access time zones (e.g. 9 AM to 5 PM Monday to Friday).
  • Track real-time door open events and access logs.
(41-9)

Troubleshooting - Reader and Lock Noise

  • Check reader voltage if card swipes fail to register.
  • Add diode across lock terminals to absorb voltage inductive spikes.
  • Use shielded cable to stop electrical noise on Wiegand data lines.
(41-10)

Practical Lab Guide

  • Wire an RFID reader, Maglock, and REX button to a 1-door panel.
  • Enroll a new user RFID card on the management software.
  • Test fire alarm override cut to ensure instant door release.
