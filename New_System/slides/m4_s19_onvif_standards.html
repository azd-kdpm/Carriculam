What is ONVIF? (Open Network Video Interface Forum)

  • Open Global Standard: Universal non-profit industry forum standardizing IP physical security device communication.
  • Multi-Vendor Compatibility: Allows Hikvision, Dahua, Axis, Hanwha, and Bosch cameras to stream seamlessly to any NVR/VMS.
  • Vendor Lock-In Prevention: Protects end-users from being trapped in proprietary single-brand hardware ecosystems.
Industry Rule: ALWAYS specify ONVIF-conformant IP cameras to guarantee future multi-brand system integration.
(20-3)

ONVIF Profile S (Basic IP Video & Audio Streaming)

  • Profile S Core Features: Standardizes H.264/H.265 video streaming, bi-directional audio, and PTZ pan/tilt/zoom controls.
  • Relay Outputs: Controls digital alarm relay outputs (e.g. strobe lights or door unlocks) over ONVIF Profile S commands.
  • Baseline Compatibility: The baseline ONVIF profile supported by virtually 100% of modern IP security cameras.
Baseline Profile: Profile S is the mandatory minimum profile for standard IP camera video streaming.
(20-4)

ONVIF Profile G (Edge SD Card Recording & Retrieval)

  • Edge Recording Control: Configures and manages local MicroSD card recording schedules on the camera itself.
  • Remote Video Retrieval: Allows VMS software to search, request, and pull recorded video clips directly from camera SD cards.
  • ANR Synchronization: Underpins Auto Network Replenishment (ANR) video sync after network outages.
Edge Standard: Require ONVIF Profile G when using camera MicroSD cards for edge failover recording.
(20-5)

ONVIF Profile T (Advanced H.265 Compression & Analytics)

  • H.265 Smart Codecs: Adds native support for H.265 video encoding and smart dynamic compression profiles.
  • HTTPS Secure Streaming: Mandates HTTPS TLS encryption for secure video stream transmission over untrusted networks.
  • Basic Motion Analytics: Standardizes motion detection event notifications and tampering alert payloads to VMS.
Modern Standard: Profile T replaces Profile S for 4K cameras requiring H.265 compression and HTTPS security.
(20-6)

ONVIF Profile M (AI Analytics & Metadata Interoperability)

  • AI Metadata Stream: Standardizes AI analytical metadata payloads (human attributes, vehicle license plates, face bounding boxes).
  • Cross-Brand AI Integration: Transmits Dahua/Hikvision AI camera analytics to Milestone/Genetec VMS servers without custom APIs.
  • Rule Configuration: Allows VMS platforms to configure line crossing and intrusion analytics rules directly on the camera.
AI Standard: Specify ONVIF Profile M for projects using AI video analytics and metadata search.
(20-7)

WS-Discovery Protocol & Automatic Camera Search

  • Web Services Discovery: Multicast UDP protocol (Port 3702) that automatically discovers unconfigured cameras on the local network.
  • Multicast Probe Flow: VMS broadcasts Probe message to 239.255.255.250:3702; cameras respond with device Probe Match XML.
  • Zero-Conf Setup: Allows installers to add 50 new cameras to an NVR in seconds without typing individual IP addresses.
Network Tip: Enable Multicast UDP Port 3702 on managed switches to allow WS-Discovery camera auto-detection.
(20-8)

Multi-Vendor Integration Troubleshooting (Hikvision / Dahua)

  • Dedicated ONVIF User: Hikvision cameras require creating a dedicated ONVIF user account in camera security settings.
  • Time Synchronization: Camera NTP clock and NVR time MUST be synchronized within 5 seconds; time drift breaks ONVIF auth tokens.
  • Digest Authentication: Ensure ONVIF Digest / WS-Security authentication matches between camera and recorder.
Troubleshooting Fix: 90% of ONVIF connection failures are caused by incorrect camera NTP time drift or missing ONVIF user accounts.
(20-9)

ONVIF Device Test Tool & Conformance Verification

  • Official Test Tool: Automated test software issued by ONVIF to verify strict protocol specification compliance.
  • Declaration of Conformance (DoC): Official vendor document proving a camera model passed official ONVIF test suites.
  • Pre-Deployment Testing: Test camera RTSP stream, PTZ latency, and alarm inputs before shipping cameras to site.
Verification Tip: Check the official ONVIF.org conformant product database before purchasing unknown brand cameras.
