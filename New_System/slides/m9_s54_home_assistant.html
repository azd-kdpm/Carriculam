Hardware Deployment: Raspberry Pi 4 vs x86-64 Mini PC

  • Raspberry Pi 4 / 5 (4GB RAM + NVMe SSD): Low power 5W server, ideal for entry-level home setups; avoid micro-SD cards to prevent database corruption.
  • x86-64 Mini PC (Intel N100 / NUC): High-performance server for large installations running local AI face recognition (Frigate NVR) and 500+ entities.
  • Home Assistant Operating System (HA OS): Turnkey bare-metal Linux OS providing automated Supervisor add-on backups.
Hardware Tip: ALWAYS use an M.2 NVMe SSD instead of micro-SD cards to prevent database corruption from constant log writes.
(54-3)

Zigbee2MQTT (Z2M) vs ZHA (Zigbee Home Automation)

  • Zigbee2MQTT (Z2M Add-On): Decoupled MQTT architecture supporting 3,500+ Zigbee devices with deep cluster OTA firmware updates.
  • Zigbee Home Automation (ZHA Native): Native HA integration; faster setup for beginners but lacks advanced device tuning.
  • Mosquitto MQTT Broker: Central message broker passing Zigbee sensor events to Home Assistant core at 1ms speed.
Pro Choice: Zigbee2MQTT offers superior device compatibility, OTA updates, and network map visualizations.

YAML Automation Engine: Triggers, Conditions & Actions

  • Trigger (When Event Occurs): Motion detected by PIR sensor or solar power drops below 500W.
  • Condition (Only If True): Sun is below horizon AND master bedroom TV is OFF.
  • Action (Do This Task): Turn ON hallway spotlights to 30% warm white brightness (2700K).
YAML Structure: Automation logic follows a strict 3-part syntax: Trigger (When) → Condition (If) → Action (Do).

Remote Access: Nabu Casa Cloud vs WireGuard VPN & Cloudflare

  • Nabu Casa Cloud (.50/mo): 1-click encrypted remote access URL supporting Alexa/Google voice integration natively.
  • WireGuard / Tailscale VPN (Free): Creates encrypted private tunnel from smartphone to home server without opening router ports.
  • Cloudflare Tunnel + SSL: Free secure HTTPS reverse proxy with DDNS and Web Application Firewall (WAF) protection.
Security Rule: NEVER open raw port 8123 on your home router; use Nabu Casa, Cloudflare Tunnels, or Tailscale VPN.
(54-9)

Troubleshooting Database Growth & Recorder Optimization

  • Database Inflation (home-assistant_v2.db): Noisy power meter entities writing state changes every 0.1s bloats DB to 50GB.
  • Recorder Filter Exclusion: Exclude noisy entities (power, RSSI, time) in configuration.yaml under recorder: exclude.
  • Purge Auto-Keep Days: Set purge_keep_days: 7 to keep database size compact under 2GB.
Database Fix: Exclude rapidly updating energy entities from the recorder to keep Home Assistant fast and responsive.
